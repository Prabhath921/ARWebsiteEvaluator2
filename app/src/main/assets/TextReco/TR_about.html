The Evaluate Home Page feature shows dynamic recognition of Basic content that should be available in a web site's Home Page within a specified region of the camera frame.

Key Functionality

  • Detection and tracking Basic UI elements and user friendly recognised word list that should be available in a Home page

Instructions

  • Point camera at screen to evaluate
  • Single tap to focus
  • Double tap to access options menu
